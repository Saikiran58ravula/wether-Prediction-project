## ✨ Features

- 🔍 Search weather by **city name**

- 🌥️ **Dynamic weather icons** that update based on current conditions
- 📱 **Fully responsive** — works seamlessly on mobile, tablet & desktop
- ⚡ **Fast & lightweight** — pure vanilla JavaScript, no frameworks
- 🌐 Fetches **live data** from a Weather API

## ⚙️ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- A free API key from [OpenWeatherMap](https://openweathermap.org/api) *(or your chosen weather API)*

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Saikiran58ravula/weather-prediction-app.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd weather-prediction-app
   ```

3. **Add your API key**

   Open `script.js` and replace the placeholder with your API key:
   ```javascript
   const API_KEY = "your_api_key_here";
   ```

4. **Run the app**

   Simply open `index.html` in your browser — no server setup required!

## 🔌 API Reference

This app uses a **Weather API** to fetch real-time weather data.

**Sample API Call:**
```
https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}&units=metric
```

**Data retrieved:**
- Temperature (°C)
- Humidity (%)
- Wind Speed (km/h)
- Weather condition & description
- Weather icon
